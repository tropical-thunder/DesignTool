# 概要
Node.js + TypeScript + Webpackで作成した描画ツールです。
Canvasを用いた描画処理と、モジュール分割を意識した設計になっています。
TypeScriptによる型安全性と、拡張性のある構成を意識して実装しました。

## 機能一覧
・図形の描画
・図形選択機能
・描画済み図形の保持
・図形移動
・図形の削除
・図形編集メニューの表示
・図形の色の変更

## 設計意図
src/配下で、図形描画の基底クラスと、
それを継承した各図形用クラスを分離しています。

共通処理（図形移動・選択状態など）を基底クラスに集約することで、
保守性と拡張性を確保し、新しい図形を追加しやすい構成にしました。
